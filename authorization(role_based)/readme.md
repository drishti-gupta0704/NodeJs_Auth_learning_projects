#  Role-Based Authorization 

This project demonstrates Role-Based Access Control (RBAC) using Node.js, Express, and JWT authentication.

The system restricts access to certain routes based on user roles such as:
-  USER
-  ADMIN
Only users with appropriate roles can access protected resources.
